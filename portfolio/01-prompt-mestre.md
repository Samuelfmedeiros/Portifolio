# 🛰️ Prompt Mestre — Mission Control Portfolio

> Use este prompt em qualquer IA de código (Cursor, Claude, Gemini) para gerar a estrutura inicial do projeto.

---

Aja como um desenvolvedor Full Stack Sênior. Vamos criar um Portfólio Profissional chamado **'Mission Control'**.

**Perfil do Profissional:** Samuel Andrade, Analista de Dados e Produto especializado em BI, SQL e Machine Learning.

**Estética:** Ficção científica moderna (Mission Control/HUD). Deve usar efeito Parallax suave no fundo (espaço profundo).

**Temas:** 'Night Vision' (Dark Mode com preto puro e ciano neon) e 'Daylight Ops' (Light Mode cinza NASA e azul marinho).

**Tecnologias:** Next.js (App Router), Tailwind CSS, Framer Motion, Lucide React e Supabase.

**Funcionalidades Principais:**
1. **Hero Section:** HUD animado com telemetria de scroll.
2. **Hangar de Projetos:** Cards que buscam dados da API do GitHub. Destaque para o projeto 'DogWalk' e Dashboards de Dados (ANA).
3. **Utility Deck (Widgets):** Um painel lateral ou modal com: relógio de missão, calculadora de dados funcional e um mini-game de lógica.
4. **Terminal Interativo:** Um componente onde comandos como `ls projects` listam os repositórios.
5. **Integração Supabase:** Formulário de contato (Transmissão) que salva mensagens no banco de dados.

Gere a estrutura de pastas inicial, o arquivo de configuração do Tailwind para o tema e um componente de exemplo para o Widget de Vidro (Glassmorphism).
